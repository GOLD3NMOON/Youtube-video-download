### YouTube Video Downloader

#### Installation:

Clone this repository and install the required dependencies from `requirements.txt`.

```bash
pip install -r requirements.txt
```

#### How to Use:

1. Run the script providing the YouTube video link as an argument.

```bash
python main.py
```
